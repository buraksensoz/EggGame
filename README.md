# EggGame

Unity egg-jump prototype.

## Setup
- Open this folder with Unity Hub.
- Use the project Unity version specified by the project metadata.

## Gameplay
- Click/tap to jump between baskets.
- Falling to EndOfLife respawns on the last saved basket position.

## Repository Notes
- Unity-generated folders/files are ignored via `.gitignore`.
- Archive files like `.rar` are also ignored.
